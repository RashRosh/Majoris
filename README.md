# MADJORIS — сайт (прототип)

VR-разработка, 3D-моделирование, цифровые двойники и нейросети для промышленности,
строительства, медицины и образования.

## Структура
- `index.html` — весь сайт (одностраничное приложение с хеш-маршрутами: `#/`, `#/about`, `#/vr`, `#/3d`, `#/ai`, `#/partners`, `#/contacts`).
- `assets/img/` — изображения (сейчас фирменные заглушки, заменяются на реальные фото/видео заказчика).
- `favicon.svg`, `apple-touch-icon.png`, `og-cover.jpg` — иконки и превью для соцсетей.
- `robots.txt`, `sitemap.xml` — SEO.

## Hero
3D VR-шлем построен процедурно на Three.js (CDN r128). При `prefers-reduced-motion`
вращение отключается; при отсутствии WebGL показывается `assets/img/hero-poster.jpg`.

## Деплой
GitHub Pages: положить содержимое в ветку (`main` или `gh-pages`), включить Pages —
сайт работает как статика, сборка не нужна.
